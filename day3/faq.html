Frequently Asked Questions

What is HTML?

HTML, or HyperText Markup Language, is a markup language that defines the structure and content of web pages

What is CSS?

CSS is used to define styles for your web pages, including the design, layout and variations in display for different devices and screen sizes.

What is Java Script?

JavaScript (JS) is a programming language that adds interactivity to websites. It can be used to control multimedia, animate images, and create dynamically updating content.
